# Go Shop Accounting System

Simple accounting system for small shops written in Go.

## Features

- Product management
- Stock tracking
- Purchases
- Sales
- Client debts
- Debt payments
- Profit dashboard

## Stack

- Go (net/http)
- PostgreSQL
- HTML templates
- CSS

## Run project

```bash
go run .
```

Open in browser:

```
http://localhost:8080
```

## Pages

- Dashboard
- Stock
- Purchases
- Sales
- Debts
- Products
